# Random Number Generator 🎲

A React application that generates a random number using **useState** and updates the UI dynamically.

---

## 🎯 Objective

To demonstrate state updates and conditional rendering by generating random numbers.

---

## ⚙️ Features

* Generate random number (1–100)
* Display number dynamically
* Show placeholder message before generation
* Optional reset functionality

---

## 🧠 Concepts Used

* React Functional Components
* useState Hook
* Conditional Rendering
* Event Handling

---

## 📸 Behavior

* Initially shows: **"No number generated yet"**
* Clicking button ➜ generates a random number
* UI updates instantly on every click

---

## 🛠️ Tech Stack

* React JS
* JavaScript (ES6)
* HTML & CSS

---

## 🚀 How to Run

```bash
npm install
npm start
```
